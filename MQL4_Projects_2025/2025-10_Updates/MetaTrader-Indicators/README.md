# MetaTrader Trading Indicators and Expert Advisors

This repository contains a collection of MetaTrader 4 (MQL4) and MetaTrader 5 (MQL5) indicators and Expert Advisors (EAs) for trading analysis and automation.

## Contents

### Indicators
- **3LS (Multi-Timeframe + Alerts)** - 3LS indicator with multi-timeframe support and alerts
- **2 MA Trend** - Dual moving average trend indicator
- **2MA** - Two moving average crossover indicator
- **3LS+LowBand** - 3LS indicator with lower band
- **Arrows** - Arrow-based signal indicator
- **BO dashboard** - Bollinger Bands dashboard
- **CS+MA Histogram** - Currency Strength with Moving Average Histogram
- **CurrencyStrengthHistogram** - Currency strength histogram indicator
- **CurrencyStrengthHistogram_2TF** - Currency strength histogram with 2 timeframes
- **CurrencyStrengthXumaFilter** - Currency strength with Xuma filter
- **CurrencyStrengthZones** - Currency strength zone indicator
- **Pattern123** - Pattern recognition indicator
- **smLazyTMA HuskyBands** - Lazy TMA with Husky Bands indicator
- **opita-3ls-husky** - 3LS indicator with Husky bands
- **opita04 3LS with BB** - 3LS indicator with Bollinger Bands

### Expert Advisors
- **EA - Currency Strength Dashboard** - Expert Advisor for currency strength dashboard
- **EA - Currency Strength Trader** - Expert Advisor for automated currency strength trading

### MQL5 Files
- **4TF HA** - 4 Timeframe Heiken Ashi indicator
- **smLazyTMA HuskyBands_v2.1** - Lazy TMA with Husky Bands (MQL5 version)

## Installation

1. Copy the `.mq4` files to your MetaTrader 4 `MQL4/Indicators/` or `MQL4/Experts/` directory
2. Copy the `.mq5` files to your MetaTrader 5 `MQL5/Indicators/` or `MQL5/Experts/` directory
3. Compile the files in MetaEditor
4. Restart MetaTrader to load the indicators/EAs

## Usage

Each indicator/EA may have its own configuration parameters. Please refer to the individual file comments for specific usage instructions.

## Requirements

- MetaTrader 4 or MetaTrader 5
- MetaEditor for compilation
